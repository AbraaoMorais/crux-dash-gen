Crux Dashboard Generator

Gere insights de desempenho para qualquer site usando a visualização de dados Crux.

Funciona para todos os sites?

Não! Se sua origem não estiver incluída no conjunto de dados do CrUX, não haverá dados para exibir. Há mais de 15 milhões de origens no conjunto de dados, mas a que você quer pode não ter dados suficientes para ser incluído. Alguns problemas comuns com origens são fornecer o protocolo errado, por exemplo, http:// em vez de https://, e omitir o subdomínio quando necessário. Alguns sites incluem redirecionamentos. Portanto, se http://example.com redirecionar para https://www.example.com, use o último, que é a versão canônica da origem.
